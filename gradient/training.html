GRADIENT DESCENT

Gradient Descent is one of the simplest optimization algorithms. We can use it tro find the optimal values of W and b.

The Algorithm

We start with some random values of W and b. We repeatedly correct their values using these equations. $$ W = W - \alpha \frac{\partial J}{\partial W} $$ $$ b = b - \alpha \frac{\partial J}{\partial b} $$ Where \(\alpha\) is the Learning Rate (LR).

With each iteration the error \(J\) should continue to go down. As we reach the minimum value of \(J\) the corrections to W and b will be smaller and smaller.

The Derivatives

We have seen the cost function \(J\) as: $$ J = \frac{1}{2m} \sum (y - y')^{2} $$ Or, $$ J = \frac{1}{2m} \sum (y^{2} - 2yy' + y'^{2}) $$ Using the chain rule of derivatives we can write. $$ \frac{\partial J}{\partial W} = \frac{1}{2m} \sum (-2y\frac{\partial y'}{\partial W} + 2y'\frac{\partial y'}{\partial W}) $$ We know: $$ y' = W \times x + b $$ Then: $$ \frac{\partial y'}{\partial W} = x $$ Finally we can write: $$ \frac{\partial J}{\partial W} = \frac{1}{m} \sum (-yx + y'x) $$ Similarly we can work out: $$ \frac{\partial J}{\partial b} = \frac{1}{m} \sum (-y + y') $$
